# 前期风险与待确认事项

## 当前已知风险

| 风险 | 当前证据 | 处理方式 |
|---|---|---|
| WSL版本与原计划不同 | 当前为Ubuntu 24.04.1，原计划曾写22.04 | 使用`uv`隔离Python 3.10；不重装发行版，后续用真实依赖验证兼容性 |
| WSL的非交互Shell未自动找到`uv` | `uv`已安装在`~/.local/bin/uv`，交互终端可用 | 必要时将`$HOME/.local/bin`加入PATH；自动检查可使用绝对路径 |
| D盘空间有限 | 仅余约77 GiB；完整Train和Validation约53.3 GiB | 不下载完整split；正式阶段只下载约7GB的确定性子集，并按需增量扩展 |
| 尚未下载正式AV2数据 | 当前只使用约220 KB的官方测试场景 | 训练阶段再按空间预算分片下载，不在准备阶段扩大数据范围 |
| PyTorch尚未配置 | 准备阶段未安装和验证GPU版PyTorch | 准备阶段不要求；训练前按官方页面安装Linux CUDA轮子 |
| AV2地图API可能随版本变化 | 当前已验证AV2 0.3.6和官方测试场景 | 保持`av2_reader`为独立适配层，并由API入口测试尽早发现变化 |
| 30类技能中部分需要视觉数据 | 天气、眩光和相机退化无法由Motion Forecasting数据直接生成 | 仅保留分类定义，不在第一版标记为可执行技能 |
| 横穿技能缺少真实遮挡信息 | Motion Forecasting不提供摄像头可见性 | 第一版只建模横穿轨迹，不声称实现视觉遮挡后显现 |
| 本地路径含中文和空格 | 当前仓库路径包含中文与空格 | 所有命令对路径加引号；长期运行若遇到第三方工具问题，再迁移到纯ASCII路径 |

## 进入训练阶段前必须确认

- 正式数据下载前再次确认AV2版本和许可；
- 确定正式数据下载空间；
- 安装并验证GPU版PyTorch；
- 固定AV2 API、PyTorch和CUDA版本；
- 用户明确授权进入训练阶段。
